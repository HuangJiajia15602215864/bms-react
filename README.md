# bms-react
React 基础入门项目——后台管理系统

### 项目目录
```
    |
    |---src                     // 资源文件
    |    |---assets             // 图片、字体等其他静态资源文件
    |    |---components         // 公共组件
    |    |---routes             // 路由配置文件
    |    |---store              // 状态存储
    |    |---style              // 公共css文件
    |    |---utils              // 公共方法
    |    |---views              // 页面
    |    |---index.js           // 入口文件
    |
    |---craco.config.js         // 项目配置文件，因为我们不用 npm run reject将配置暴露出来
    |--- 其他文件
```

- antd
